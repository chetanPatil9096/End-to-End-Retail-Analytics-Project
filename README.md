# End-to-End Retail Analytics Project

## Project Overview
This project is an End-to-End Retail Analytics Pipeline built using Python, SQL, MySQL, and Power BI.

The project focuses on retail sales analysis, customer behavior insights, data cleaning, SQL analytics, and interactive dashboard reporting.

---

## Tools & Technologies Used

- Python
- Pandas
- NumPy
- SQL
- MySQL
- Power BI
- Jupyter Notebook
- SQLAlchemy
- PyMySQL

---

## Project Workflow

1. Data Collection
2. Data Cleaning using Python
3. Feature Engineering
4. SQL Data Analysis
5. MySQL Database Integration
6. Power BI Dashboard Development
7. Business Insights Generation

---

## Key Features

- Data Cleaning & Preprocessing
- Revenue Analysis
- Top Products Analysis
- Country-wise Sales Analysis
- Monthly Revenue Trend
- Customer Revenue Analysis
- Interactive Power BI Dashboard
- SQL-Based Business Queries

## SQL Analytics

The project includes advanced SQL queries such as:

- Top Countries by Revenue
- Monthly Revenue Trend
- Customer Lifetime Value
- Top Selling Products
- Repeat Customer Revenue Analysis

---

## Business Insights

- United Kingdom generated the highest revenue.
- Seasonal sales spikes were observed during year-end months.
- A small group of customers contributed significantly to total revenue.
- Top products generated a major portion of total sales revenue.
